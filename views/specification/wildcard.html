  1. 泛型中的通配符 T，E，K，V，？

本质上这些个都是通配符，没啥区别，换成 A-Z 之间的任何一个 字母都可以。 通常情况下，T，E，K，V，？ 是这样约定的：

  • ？ 表示不确定的 java 类型
  • T (type) 表示具体的一个 java 类型
  • K V (key value) 分别代表 java 键值中的 Key Value
  • E (element) 代表 Element
