# Vue-Demo

  • 这是在学习Vue的同时，写的一个demo

  • 项目地址：https://github.com/zpj80231/vue-demo

    点击右上角克隆或下载 下载完成之后，在vue-demo文件目录打开命令窗口 输入命令： npm install （需提前装好node.js环境，如果没有请自行百度） 待npm安装完成之后输入： npm run dev 即可在浏览器看到窗口界面
